# use-ls

Utility and integration packages for [Legend-State](https://legendapp.com/open-source/state/).

## Packages

| Package | Description |
|---------|-------------|
| [`@usels/core`](./packages/core) | React hooks for elements, sensors, and browser APIs |
| [`@usels/integrations`](./packages/integrations) | third party integration with Legend-State |

### @usels/core

```bash
pnpm add @usels/core@beta @legendapp/state@beta
```

## Development

```bash
pnpm install      # install dependencies
pnpm build        # build all packages
pnpm dev          # watch mode
pnpm test         # run tests
pnpm lint         # lint
```

## Requirements

- Node >= 22
- pnpm >= 8
